This Google Apps Script project implements a web application designed for managing fan submissions and selecting posts for Facebook posting. The application utilizes the `CCLIBRARIES` library and a JSON database structure to store and manage post data. The web application includes a user interface (UI) built with HTML, CSS, and JavaScript, and backend functionality implemented with JavaScript server-side code.

## Functionality

The web application provides the following key features:

* **Fan Submission Management:** The application enables the management of fan submissions, including their storage, display, and filtering based on compliance with guidelines.

* **Post Selection:** The application allows admins to select posts for posting to Facebook, based on their compliance and other factors.

* **Status Tracking:** The application tracks the status of posts, indicating whether they are pending, rejected, scheduled, or posted.

* **Snap to Post:** The application provides a feature that allows admins to quickly scroll to a specific post in the feed.

* **Copy to Clipboard:** The application enables admins to copy the content of a post to the clipboard for convenient sharing.

* **Database Integration:** The application uses a JSON database to store and manage post data, ensuring efficient data management.

## Components

The web application is composed of various components:

* **Client-Side UI:** The UI is built using HTML, CSS, and JavaScript. It provides a user-friendly interface for interacting with the application's features.

* **Backend Logic:** The backend logic is implemented with JavaScript server-side code, handling database operations, data processing, and communication with the UI.

* **JSON Database:** The JSON database serves as the storage for post data, allowing for efficient and scalable data management.

## Usage

The web application is designed for use by admins who are responsible for managing fan submissions and selecting posts for Facebook posting. The application provides a comprehensive set of tools for streamlining this process, improving efficiency, and ensuring compliance with guidelines. The UI is intuitive and easy to use, allowing admins to quickly and effectively manage the application's functionalities.
